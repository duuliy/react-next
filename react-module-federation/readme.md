# react + 模块联邦 + 嵌套打包
这个例子证明react是可以嵌套打包的
fe3=>fe2=>fe1

## How to run

* start fe1
```shell
cd fe1
yarn
yarn start
```

* start fe2
```shell
cd fe1
yarn
yarn start
```

* start fe3
```shell
cd fe1
yarn
yarn start
```
